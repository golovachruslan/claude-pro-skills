# Project Brief

## Overview
project-context plugin to manage claude project context

A Claude Code plugin that helps maintain project context across sessions, enabling structured planning, dependency management, and seamless work resumption.

## Goals
- Provide persistent project context for Claude Code sessions
- Enable structured planning and multi-agent execution
- Support cross-project dependency management

## Scope
### In Scope
- Project context file management (brief, architecture, state, progress, patterns)
- Session pause/resume with handoff documents
- Dependency fetching from local paths and git URLs
- Plan creation and execution via multi-agent workflows

### Out of Scope
- Non-Claude Code integrations
- Database or persistent storage beyond markdown files
